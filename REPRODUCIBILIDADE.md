Este documento detalha o ambiente de hardware, software e os parâmetros de execução utilizados para gerar os resultados de desempenho nas tarefas C (SAXPY) e D (overhead) deste projeto OpenMP. O objetivo é garantir a reprodutibilidade integral dos dados apresentados.

A coleta de dados foi realizada em um sistema host com as seguintes especificações de hardware: processador (CPU) AMD Ryzen 5 3400G, configurado com 4 Cores físicos e 8 Threads lógicas; memória RAM DDR4 de 16 GBytes; e placa de vídeo (GPU) NVIDIA GeForce GTX 1650. Os testes de desempenho foram conduzidos dentro de uma Máquina Virtual (VM) para isolamento do sistema operacional host. A Máquina Virtual utilizada rodava o sistema operacional Ubuntu 24.04.3 amd64 e foi alocada com recursos limitados a 4 GBytes de RAM e 4 Cores do processador host.

Ambiente de Software e FerramentasO sistema operacional host primário utilizado foi o Microsoft Windows 10 Home (10.0.18045 Compilação 19045). O ambiente de compilação e execução, no entanto, residiu integralmente na Máquina Virtual Ubuntu. O compilador C/C++ utilizado foi o GCC (Ubuntu 13.3.0-6ubuntu2~24.04) 13.3.0, que oferece suporte às especificações OpenMP 5.1 e 5.2. As ferramentas de automação e análise utilizaram o Python versão 3.12.3 em um ambiente virtual (venv), sendo as bibliotecas essenciais para processamento e plotagem: pandas (2.3.3), matplotlib (3.10.7) e numpy (2.3.2).

Parâmetros de ExecuçãoOs parâmetros de teste foram gerenciados pelos arquivos Makefile e run.sh. As flags de compilação incluíram otimizações (-Wall, -Wextra, -O3) e a ativação do paralelismo OpenMP (-fopenmp). Os parâmetros de teste definiram o tamanho do problema ($N$) nos valores $\{100000, 500000, 1000000\}$, o número de Threads ($T$) em $\{1, 2, 4, 8, 16\}$, com um total de 5 repetições por ponto de medição para fins de cálculo estatístico. A utilização dessas configurações em uma arquitetura virtual de 4 núcleos garante que os resultados obtidos são válidos para a análise de escalabilidade e overhead neste ambiente específico.
